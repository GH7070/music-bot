# 🎵 بوت الموسيقى على Discord\n\nبوت موسيقى احترافي لـ Discord مع دعم YouTube و Spotify والعديد من مصادر الموسيقى.\n\n## ✨ المميزات\n\n- ✅ تشغيل الموسيقى من YouTube و Spotify\n- ✅ قائمة انتظار للأغاني\n- ✅ تحكم بمستوى الصوت\n- ✅ أوامر متعددة للتشغيل والإيقاف\n- ✅ دعم 10 بوتات في نفس الوقت\n\n## 🎮 الأوامر\n\n### أوامر التشغيل\n- `!play` أو `!p` - تشغيل أغنية\n- `!Play` أو `!P` - نفس الأمر بحروف كبيرة\n\n### أوامر الإيقاف\n- `!stop` أو `!s` - إيقاف التشغيل\n- `!Stop` أو `!S` - نفس الأمر بحروف كبيرة\n- `!skip` - تخطي الأغنية\n\n### أوامر الصوت\n- `!volume` أو `!v` - تعديل مستوى الصوت (1-100)\n\n## 📦 التثبيت\n\nتابع ملف `SETUP_GUIDE.md`\n
